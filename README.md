## Melhorias para o APP

- **Animação de movimento fluida**: Adicionar animações suaves para aprimorar a experiência do usuário ao navegar no aplicativo.

- **Trocar Dio**: Substituir a biblioteca Dio por uma alternativa, já que foi descontinuada. Pesquisar e implementar uma solução mais atualizada.

- **Navigator Bar**: Implementar uma barra de navegação para permitir uma transição fácil entre as diferentes seções do aplicativo.

- **Página de pedidos realizados**: Criar uma página dedicada para mostrar os pedidos realizados, com opções para filtrar entre pedidos em andamento e pedidos concluídos.

- **Deslogar perfil de usuário**: Adicionar funcionalidade para permitir que os usuários façam logout de seus perfis no aplicativo.

## Contribuição

Se você deseja contribuir para este projeto, siga as etapas abaixo:

1. Faça um fork deste repositório.

2. Crie uma branch para suas alterações:

   ```
   git checkout -b minha-branch
   ```

3. Faça as alterações desejadas e adicione os arquivos modificados:

   ```
   git add .
   ```

4. Faça um commit das suas alterações:

   ```
   git commit -m "Minhas alterações"
   ```

5. Faça um push para o repositório remoto:

   ```
   git push origin minha-branch
   ```

6. Abra um pull request no repositório original.
